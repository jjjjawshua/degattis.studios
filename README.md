# degattis.studios

A minimalist portfolio and shop website for contemporary art.

## About

Clean, black and white website featuring a rotating logo, portfolio collection, online shop, and contact form. Built with pure HTML, CSS, and vanilla JavaScript.

## Features

- Minimalist monochrome design
- Animated spinning logo
- Single-page navigation
- Portfolio/collection gallery
- Shop with product listings
- About page
- Contact form
- Responsive layout
- Social media links (email & Instagram)

## Quick Start

1. Rename `degattis-studios.html` to `index.html`
2. Upload to your hosting platform
3. Done!

## Deployment Options

### Netlify (Recommended)
1. Go to https://app.netlify.com/drop
2. Drag and drop `index.html`
3. Get instant live URL

### GitHub Pages
1. Create repository
2. Upload `index.html`
3. Enable Pages in Settings

### Vercel
1. Sign up at https://vercel.com
2. Deploy via drag-and-drop

## Customization

### Update Content
- **About section**: Edit the text in the `#about` page section
- **Shop items**: Modify the `.shop-item` divs with your products and prices
- **Collection**: Update `.gallery-item` divs with your artwork
- **Contact info**: Change email and Instagram links in footer

### Add Your Images
Replace placeholder text in gallery/shop items:
```html
<div class="shop-item-image">
    <img src="your-image.jpg" alt="artwork description">
</div>
```

### Change Colors
Currently black (#000000) and white (#ffffff). Edit in the CSS section.
